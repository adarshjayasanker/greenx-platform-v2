### Greenx Platform v2.0

## Overview

Greenx Platform v2 is the next-generation web platform for Greenx Pest Control Services.
It is being rebuilt from the ground up with a modern JavaScript stack, focusing on performance,
maintainability, security, and scalability.

The platform serves two primary purposes:

- A modern customer-facing website that generates and manages service enquiries.
- A secure administration panel for managing business operations.

The project follows clean architecture principles and is designed to evolve incrementally. The 
technology stack can be upgraded over time without requiring a complete rewrite.

## Project Goals

- Build a fast and responsive web application.
- Deliver a reliable production-ready system.
- Maintain clean, modular, and scalable code.
- Follow professional software engineering practices.
- Create an architecture that supports future enhancements.

## Architecture Principles

The project is built around the following principles:

- Separation of concerns
- Modular architecture
- Reusable components
- Centralized configuration
- Clean API design
- Security by default
- Performance-conscious implementation
- Incremental evolution

The architecture is intentionally designed so individual technologies can be replaced or upgraded with
minimal impact on the rest of the system.
